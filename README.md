# finalProject01
# 👩‍⚕진료이즈백 EMR 프로젝트

## 📋프로젝트 소개
병원 관리 시스템을 구현한 프로젝트입니다.
<br>

## ⏱개발 기간
23.09.07 ~ 23.10.16

## 👥 멤버 구성
* 김예린 - 직원 관리, 발주, 공지사항, 캘린더, 통계
* 윤강혁 - 환자 관리 및 접수, 제증명, 문자, 입퇴원 수속
* 이경민 - 로그인, 알림, 물리치료
* 정수지 - 처방 및 처치, 병동 관리, 소모품 발주 신청, 엑스레이 파일 등록
* 정은비 - 환자 진료차트 작성, 제증명

## ⚙개발 환경
* Java 8
* JDK 1.8.0
* IDE: STS 3.9
* Framework : Spring(5.3.x)
* DataBase : Oracle DB(11xe)
* ORM : MyBatis
* 협업 툴 : SVN
